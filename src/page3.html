UI: User Interface

UI Designers chose visual design elements like color pallets, fonts, and organize page layouts.
Goal: Make apps and website appealing and easy to navigate.
They Design interactive elements like:

  • Scrollers
  • Buttons
  • Toggles
  • Text Fields
  • Drop down menus

They work with UX Designers and web Developers



Steps in the UI process:

  1. Access end user needs
  2. Examine features
  3. Make Sketches
  4. Design Components
  5. Build a prototype
  6. Redo when need to
